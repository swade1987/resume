
# swade1987.github.io/resume

[![Build Status](https://secure.travis-ci.org/biomadeira/sustain.png?branch=gh-pages)](https://travis-ci.org/swade1987/resume)

Personal homepage (curriculum vitae or resume) based on [Jo√£o's](https://github.com/joaomoreno/resume).
Vitae is built with [skeleton](http://getskeleton.com/), powered by [Jekyll](http://jekyllrb.com/) and freely
hosted in [Github pages](https://pages.github.com/).

## How to use it

Simply clone/fork this repository, and then run the following commands from inside the directory.

```bash
$ gem install jekyll
$ jekyll serve
```

Edit the site attributes in *_config.yml* and edit the various entries in *_includes/* and *_posts/*.
